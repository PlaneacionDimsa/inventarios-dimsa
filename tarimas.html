Configurar Firebase (sync en vivo)

Sigue estos pasos para que aduana y tú vean el mismo inventario en tiempo real:

  1. Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google.
  2. Haz clic en "Agregar proyecto" → ponle nombre (ej. tarimas-planta) → crea.
  3. En el panel del proyecto, selecciona Firestore Database → "Crear base de datos" → modo producción → elige la región más cercana.
  4. En Reglas de Firestore, pega esto para permitir acceso:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{db}/documents/{doc=**} {
        allow read, write: if true;
      }
    }
  5. Ve a Configuración del proyecto (⚙️) → pestaña "General" → sección "Tus apps" → agrega una app web (</>).
  6. Copia el objeto firebaseConfig que aparece y pégalo en el archivo index.html en la sección FIREBASE CONFIG marcada abajo.
  7. Guarda el archivo y ábrelo desde un servidor (puedes usar Netlify Drop — arrastra la carpeta a app.netlify.com/drop — gratis).

Una vez configurado, cualquiera que abra el link verá el inventario actualizado al segundo.
